HOW TO CREATE AND PUSH THIS PROJECT TO GITHUB (STEP-BY-STEP)
================================================================================

STEP 1 — Create the GitHub Repository
--------------------------------------
1. Go to https://github.com and sign in to your account.
2. Click the "+" icon (top-right) → "New repository".
3. Fill in the form:
     Repository name:  AIGT-Pattern-Detection
     Description:      Adaptive AI-Generated Text Detection via Generative Pattern of LLM
     Visibility:       Public   ← IMPORTANT: must be public for submission
     Add README:       OFF  (we already have one)
     Add .gitignore:   OFF  (we already have one)
     Add license:      OFF  (optional)
4. Click "Create repository".
5. Copy the repository URL shown on the next page:
     https://github.com/YOUR_USERNAME/AIGT-Pattern-Detection.git


STEP 2 — Install Git (if not already installed)
-------------------------------------------------
Download from: https://git-scm.com/downloads
Verify installation:
    git --version


STEP 3 — Configure Git Identity (first time only)
--------------------------------------------------
    git config --global user.name "Your Name"
    git config --global user.email "your_email@example.com"


STEP 4 — Initialise and Push the Project
------------------------------------------
Open a terminal in the AIGT-Pattern-Detection folder, then run:

    git init
    git add .
    git commit -m "Initial commit: full implementation of AIGT pattern-based detection framework"
    git branch -M main
    git remote add origin https://github.com/YOUR_USERNAME/AIGT-Pattern-Detection.git
    git push -u origin main

You will be prompted for your GitHub username and password.
If you use 2FA, use a Personal Access Token instead of your password:
    Go to GitHub → Settings → Developer settings → Personal access tokens → Generate new token
    Select scope: "repo" → Generate → copy the token → use as password


STEP 5 — Verify
----------------
Open your browser and navigate to:
    https://github.com/YOUR_USERNAME/AIGT-Pattern-Detection

You should see:
  ✔ notebooks/   (3 Jupyter notebooks)
  ✔ docs/        (DATASETS.md)
  ✔ README.md

  ✔ .gitignore

Replace "YOUR_USERNAME" at the top of this file with your actual GitHub username,

================================================================================
REPRODUCIBILITY NOTES
================================================================================

- All random seeds are fixed to 42 throughout the codebase.
- Train/test splits are text-disjoint (verified with assertion checks in notebook 1).
- Pattern training sets use exactly 44–50 samples per dataset as described in the thesis.
- GPU is recommended for notebooks 02 and 03 (feature extraction uses GPT-2 and
  sentence-transformers; model training uses BERT/RoBERTa fine-tuning).
- CPU-only mode is supported but will be significantly slower (~5–10x).
- Full end-to-end runtime on GPU: approximately 3–5 hours for all 5 datasets.
